# Mini Robô Sumô - Arduino Day 2023 🤖🥊

![Circuito dos Robôs](./assets/sumostrike.png)

### Componentes utilizados ⚙️
- 1 Arduino Nano
- 2 Motores DC com caixa de Redução
- 1 Driver Motor Ponte H
- 1 Módulo Bluetooth HC-05
